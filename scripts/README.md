# Development & Debug Scripts

Run from project root after `pip install -e .`:

- `python scripts/find_elements.py` - Inspect Runi portal elements (XPath discovery)
- `python scripts/check_login_fields.py` - List login form fields
- `python scripts/debug_after_login.py` - Debug post-login flow
- `python scripts/test_telegram.py` - Verify Telegram bot configuration
